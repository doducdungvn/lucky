📖 Hướng Dẫn Nhập Tắt

Bạn có thể sử dụng các ký tự viết tắt dưới đây. Hệ thống tự động nhận diện cả các tên thay thế (ví dụ: dau5 tương đương d5, dit5 tương đương d'5, he05/bo05 tương đương h05, tb tương đương tobe, dc/dl tương đương dauchan/daule, v.v.):

🔥 Nhóm Kép & Đặc biệt
  • k: Kép bằng (00, 11, ..., 99)
  • kl: Kép lệch (05, 50, 16, 61, ...)
  • apkep: Sát kép (01, 10, 12, 21, ...)
  • kepam: Kép âm (07, 70, 14, 41, ...)
  • tongchan / tongle: Tổng chẵn / Tổng lẻ
  • tongto / tongbe: Tổng to / Tổng bé
  • cc / cl / lc / ll: Chẵn Chẵn / Chẵn Lẻ / Lẻ Chẵn / Lẻ Lẻ
  • toto / tobe / beto / bebe: To To / To Bé / Bé To / Bé Bé
📊 Đầu, Đuôi & Chạm
  • d0 - d9: Đầu 0 đến Đầu 9
  • d'0 - d'9: Đuôi 0 đến Đuôi 9
  • cham0 - cham9: Chạm 0 đến Chạm 9
  • dauto / daube: Đầu to (5-9) / Đầu bé (0-4)
  • dauchan / daule: Đầu chẵn / Đầu lẻ
  • ditchan / ditle: Đuôi chẵn / Đuôi lẻ
  • ditto / ditbe: Đuôi to / Đuôi bé
🎯 Tổng Đề (t0 - t9)
  • t0 - t9: Tổng 2 số chia 10 dư từ 0 đến 9. Ví dụ: t0 (00, 19, 28, 37, 46, ...)
⭐ Hệ / Bộ số (h00 - h99)
  • h00: 00, 55, 05, 50
  • h11: 11, 66, 16, 61
  • h22: 22, 77, 27, 72
  • h33: 33, 88, 38, 83
  • h44: 44, 99, 49, 94
  • h12: 12, 21, 17, 71, 62, 26, 67, 76
    • Tất cả các bộ số hệ khác từ h01 đến h99 đều được hỗ trợ đầy đủ.
