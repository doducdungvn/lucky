Cấu hình GitHub

Bảo mật

Token này được lưu an toàn tại localStorage trình duyệt của bạn.

💡 Hướng dẫn kết nối GitHub

1. Truy cập GitHub của bạn và tạo 1 Token cá nhân (Classic PAT hoặc Fine-grained) với quyền ghi **contents/write** cho repo này.
2. Điền Token và thông tin Repository của bạn vào bảng cấu hình bên trên.
3. Nhấp "Lưu cấu hình", sau đó bạn có thể thêm/sửa/xóa trực tiếp và đẩy các thay đổi lên GitHub chỉ bằng 1 cú click!
