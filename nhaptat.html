Bảo mật

Lưu an toàn trong localStorage trình duyệt.

Hướng dẫn kết nối GitHub

1. Tạo 1 Personal Access Token (PAT) trên GitHub với quyền ghi contents/write.
2. Nhập Token & thông tin Repository vào bảng trên.
3. Nhấp "Lưu cấu hình", bạn có thể Thêm/Sửa/Xóa viết tắt và đẩy trực tiếp lên GitHub!
